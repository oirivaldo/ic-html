O que é um computador?

Um computador é uma máquina eletrônica capaz de processar informações seguindo instruções de programas.
Segundo Von Neumann, sua arquitetura básica inclui:

  • Unidade de Processamento Central (CPU)
  • Memória Principal
  • Dispositivos de Entrada/Saída

Componentes Básicos

Todo sistema computacional moderno possui:

Hardware: Parte física (processador, memória RAM, HD)
Software: Programas e sistemas operacionais
Peopleware: Usuários e profissionais

Importância da Computação

A computação revolucionou a sociedade moderna, sendo essencial em áreas como:

- Medicina
- Engenharia
- Comunicação
- Pesquisa Científica

O que são entidades HTML?

Utilizar alguns caracteres pode ser bastante trabalhoso, como no caso de < (escrito com &lt;, &#60; ou &#x3c;). Para isso, utilizamos &amp; para representar o próprio "&". Veja exemplos na tabela:

Exemplos de Entidades

Caractere Entidade Nomeada Entidade Decimal Entidade Hexadecimal Uso
< &lt; &#60; &#x3c; Tag HTML
& &amp; &#38; &#x26; E comercial isolado
> &gt; &#62; &#x3e; Fechamento de tags
" &quot; &#34; &#x22; Aspas em atributos
' &apos; &#39; &#x27; Apostrofo (XML)

Caso Especial: & Sozinho

Para exibir um "&" isolado:
&amp; → &
&#38; → &
&#x26; → &

Exemplo Complexo


    <div>
        &lt;h1&gt;Título&lt;/h1&gt;
        &lt;p&gt;Texto com &amp; e &lt;tags&gt;&lt;/p&gt;
    </div>

Renderiza como:

<h1>Título</h1>
<p>Texto com & e <tags></p>

Boas Práticas

  • Sempre termine entidades com ;
  • Prefira entidades nomeadas para melhor legibilidade (&lt; vs &#60;)
  • Em conteúdo dinâmico, sanitize entidades para prevenir XSS

Curiosidade

Para exibir as entidades literalmente (como texto):
&amp;amp; → &amp;
&amp;#60; → &#60;

O que são comentários e como posso utilizá-los em HTML, JavaScript e CSS?

Comentários podem ser muito úteis para dar orientações ou lembretes para todos aqueles que acessarem o código. Mas dependendo de qual linguagem, seja essa linguagem de marcação ou programação, a forma de se comentar pode variar. E isso ocorre com o HTML, CSS e JavaScript. Vamos conferir a próxima tabela:

Linguagem Comentário de Linha Única Comentário de Múltiplas Linhas Exemplo Prático
HTML Não possui <!-- Comentário --> <!-- Menu principal -->
<nav>...</nav>
CSS /* Comentário */ /* Comentário
Multi-linha */		 .botao {
  color: red; /* Cor antiga */
}
JavaScript // Comentário /* Comentário
Multi-linha */		 const preco = 99.90; // Preço em reais

Exemplos Detalhados

HTML
<!--
        Bloco comentado:
        <div class="old-version">
            Conteúdo antigo
        </div>
    -->
    <header>
        <!-- Logo principal -->
        <img src="logo.png">
    </header>
CSS
/* Estilo da navbar */
    nav {
        padding: 1rem;
        /*
        background: blue; - Cor alterada em 01/01
        */
        background: #2e2e2e;
    }
JavaScript
function calcularTotal() {
        /* 
        Fórmula antiga:
        return quantidade * preco * 1.1;
        */

        // Nova fórmula com desconto progressivo
        return quantidade * (preco - desconto);
    }

⚠️ Cuidados Importantes

  • Em HTML, evite comentários aninhados: <!-- <!-- Comentário --> -->
  • No CSS, // Comentário só funciona em pré-processadores como Sass
  • Em JS, comentários não fechados /* ... quebram o código
  • Nunca deixe dados sensíveis em comentários (senhas, chaves de API)

💡 Dicas de Utilização

  • Use para documentar funções complexas
  • Mantenha comentários breves e objetivos
  • Utilize tags como TODO ou FIXME para marcações:
    <!-- TODO: Atualizar link -->
  • Em IDEs, use atalhos:
    • Ctrl + / (Windows) ou Cmd + / (Mac)
    • Shift + Alt + A no VS Code para blocos
